01SETUP · ONE TIME

Get
set up

A few one-time steps per machine: install Nix, then get on the private Tailscale network.

1

Install Nix

xyne-boxes runs entirely through Nix, so you need it first. Don't have it yet? Install it in one step with nixone.

2

Install Tailscale

Grab the app from the Mac App Store. Then check the CLI works — tailscale --help should print something.

Terminal says command not found?

The app doesn't always add the CLI to your PATH. Paste this once to create a wrapper that points at it:

$mkdir -p /usr/local/bin
printf '#!/bin/sh\nexec /Applications/Tailscale.app/Contents/MacOS/Tailscale "$@"\n' > /usr/local/bin/tailscale
chmod +x /usr/local/bin/tailscale
3

Join the network

Connect your machine to the VPN. A browser window opens for you to sign in.

$sudo tailscale up --login-server=https://headscale.nixos.asia --hostname $(hostname -s)
02EVERYDAY USE

Create a box
& jump in

This is the whole workflow. Pick a name, and you're inside.

1

Create it

Pick any name for your box. This builds a fresh Linux container with a ready-to-go dev environment inside.

The first time you run it — and roughly once a week after — you'll sign in: the terminal prints a link and a short code. Open google.com/device, enter the code, and sign in with your Juspay Google account. That grants your SSH access — no keys to manage.

$nix run https://github.com/juspay/xyne-boxes/archive/main.zip create <name>
2

Jump in

Load your SSH key into the agent so it can be forwarded into the box.

$eval "$(ssh-agent -s)"
ssh-add ~/.ssh/id_ed25519

Then connect. This drops you into a shell on the box as the toor user.

$nix run https://github.com/juspay/xyne-boxes/archive/main.zip connect <name>
Prefer to type ssh <name>?

Do this once and plain ssh <name> works for every box you create.

Let SSH pick up each box's config

$mkdir -p ~/.ssh
grep -qxF 'Include ~/.pu-state/*/ssh_config' ~/.ssh/config 2>/dev/null \
  || printf '\nInclude ~/.pu-state/*/ssh_config\n' >> ~/.ssh/config

Now just run ssh <name> to connect.

03INSIDE THE BOX

What
you get

Every box is an Incus container cloned from a loaded NixOS base image — not a bare machine.

Real dev env

git, vim, tmux, direnv, starship and more, ready to go. VS Code Remote works out of the box.

Warm Nix

Flakes are on and the box shares the host's /nix/store — dev shells are fast and common Juspay dev-envs are pre-cached.

Persistent

You land as toor with your own home. It survives stop/start — gone only when you run destroy.

Extras

list shows your boxes, fork <source> <name> clones one, and destroy <name> removes it.

New to Nix? Start at nixos.asia/nix-first to learn how to install packages and work in the environment. Ready for the next step? Set up home-manager with the nixos-unified-template.
04GOOD TO KNOW

When access
expires

Your access lasts about a week. When it lapses, reconnecting fails with an error like this:

pu@pu: Permission denied (publickey,keyboard-interactive). Connection closed by UNKNOWN port 65535

Nothing's broken — just run connect again and sign in the same way you did the first time. About 20 seconds, and you're good for another week.

$nix run https://github.com/juspay/xyne-boxes/archive/main.zip connect
Seeing a GitHub 403 error? Turn the VPN off and try again. If it keeps happening, set up a GitHub token for a higher rate limit.
